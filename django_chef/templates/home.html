About DjangoChef

DjangoChef is a simple, open-source web app built with Django that helps you manage recipes and plan your meals. It’s designed as both a personal project and a portfolio piece showcasing clean architecture, CRUD functionality, and a responsive UI.

You can create, edit, view, and delete recipes, categorize them, and organize your weekly meal plans. It’s perfect for keeping your favorite dishes in one place and planning your cooking schedule.

DjangoChef is built with Django and Python on the backend, using HTML, CSS (Tailwind or Bootstrap), and JavaScript on the frontend. It follows the MVC pattern and demonstrates full CRUD (Create, Read, Update, Delete) operations for recipes and meal plans.

It’s for anyone who loves to cook, wants a simple way to organize recipes, or is learning web development and wants to see a Django CRUD project in action.

DjangoChef was created as a hands-on learning project to strengthen Django skills and demonstrate full-stack development — from database models and views to user experience and design.
